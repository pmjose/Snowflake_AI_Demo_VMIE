# CityFibre HR Overview (Demo)

## Workforce Focus
- Roles: network build/delivery, field engineering, NOC, partner success, sales, marketing, finance, HR, product.
- Skills: fibre build (civils, splicing), network operations, backhaul design, cloud connectivity, partner management.

## Policies (high-level demo summary)
- Safety: fibre build site safety standards; required PPE and contractor compliance.
- Training: onboarding for network build, partner success, and support tooling; refresher on safety and data handling.
- Performance: objectives tied to build velocity, uptime, partner satisfaction (NPS), install lead times.
- Flex: hybrid/field roles vary by function; critical NOC/build roles on-site/field as needed.

## Diversity & Inclusion
- Equal opportunity hiring; inclusive language in postings.
- Promote apprenticeships and local hiring in build regions.
