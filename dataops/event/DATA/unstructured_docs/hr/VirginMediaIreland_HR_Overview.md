# Virgin Media Ireland HR Overview (Demo)

- Workforce spans network engineering, field operations, customer care, sales, marketing, technology, and corporate functions.  
- Culture priorities: customer-first support, safety in field operations, inclusion, and development through Liberty Global programs.  
- Training: network build standards, fibre install quality, in-home WiFi optimisation, and converged product support (broadband, TV, mobile).  
- KPIs: field quality scores, first-time-fix, customer satisfaction, adherence to safety protocols, and talent retention.  
- Benefits highlight: hybrid work options, learning budgets, and recognition for service excellence.
