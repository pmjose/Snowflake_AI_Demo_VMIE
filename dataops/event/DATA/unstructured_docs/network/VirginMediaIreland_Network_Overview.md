# Virgin Media Ireland Network Overview (Demo)

- Nationwide cable and fibre network delivering broadband, TV and phone; part of Liberty Global.  
- Rolling out full-fibre with advertised speeds up to 5Gb and improved reliability versus legacy FTTC.  
- Focus cities and regions include Dublin, Cork, Limerick, Galway, Waterford, and expanding towns.  
- WiFi Guarantee and mesh options to strengthen in-home coverage; multiroom TV supported.  
- Resilience: backbone diversity, DOCSIS upgrades, and fibre builds to reduce latency and jitter for streaming and gaming.  
- Mobile: Virgin Media Ireland operates as an MVNO (hosted on partner MNO); mobile traffic handed off via MVNO host interconnects/backhaul, not VMI-owned RAN.  
- B2B services include dedicated internet access, Ethernet, and managed connectivity for SMEs and public sector sites.  
- Competitors: eir fibre, SIRO/altnets, Sky; positioning on speed, bundled value, and service quality.  
- Demo content only; performance figures are illustrative.
