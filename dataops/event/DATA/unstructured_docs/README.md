# CityFibre Unstructured Documents (Demo)

This folder contains CityFibre-branded reference material used by Cortex Search in this demo. Content is synthesized for demonstration and draws on the included official reports (`CFIH-Group-2024-Accounts.pdf`, `CityFibre-Mid-year-update.pdf`) and public website facts (https://cityfibre.com/). Do not treat as production data.

## Folders
- finance/: financial highlights, funding, revenue mix
- strategy/: rollout goals, partner/segment priorities
- network/: footprint, performance, resilience
- sales/: wholesale partner guidance, offers
- marketing/: messaging, campaign themes
- hr/: people policy summary
