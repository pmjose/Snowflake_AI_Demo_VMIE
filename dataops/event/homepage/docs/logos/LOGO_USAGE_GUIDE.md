# Virgin Media Ireland Logos for Snowflake Intelligence

This folder contains the Virgin Media Ireland logo files needed to customize your Snowflake Intelligence interface.

## Files Available

### 1. Compact Logo (Icon/Avatar)
- **celcomdigi_logo.png** (5.8KB)
  - Compact version with Virgin Media Ireland butterfly icon
  - Blue and yellow brand colors
  - **Use for:** Icon field, agent avatar, small UI elements
  - **Dimensions:** Square/compact format

### 2. Full-Length Logo (Header/Banner)
- **celcomdigi_logo_full.png** (16KB)
  - Full-length Virgin Media Ireland logo
  - Blue and yellow brand colors
  - **Use for:** Logo field, header, banner areas
  - **Dimensions:** Wide/horizontal format

## How to Use

### Download the Logos

**Method 1: From Documentation (Recommended)**
1. Open the lab documentation
2. Navigate to "Configure Snowflake Intelligence" section
3. **Right-click** on each logo image
4. Select "Save Image As..." or "Download Image"
5. Save both files to your local machine

**Method 2: From This Directory**
1. Right-click on `celcomdigi_logo.png`
2. Select "Download" or "Save As"
3. Repeat for `celcomdigi_logo_full.png`

### Upload to Snowflake Intelligence

1. Navigate to **AI & ML** → **Agents** → **Snowflake Intelligence** tab
2. Click **Open Settings**
3. Scroll to **Custom Appearance** section

**For the Logo Field (Header/Banner):**
- Click **Upload** for Logo field
- Select **celcomdigi_logo_full.png** (full-length version)

**For the Icon Field (Avatar):**
- Click **Upload** for Icon field  
- Select **celcomdigi_logo.png** (compact version)

### Apply Virgin Media Ireland Brand Colors

**Primary Color:** `#0078D7` (Blue)
- Main buttons, highlights, primary UI elements

**Secondary Color:** `#FFCC00` (Yellow)
- Accents, secondary elements, emphasis

## Logo Usage Guide

### Compact Logo (celcomdigi_logo.png)
✅ **Use for:**
- Agent avatar/icon
- Small UI badges
- Mobile displays
- Compact spaces
- Chat conversation bubbles

❌ **Don't use for:**
- Wide header areas
- Large banner spaces
- Full-width displays

### Full-Length Logo (celcomdigi_logo_full.png)
✅ **Use for:**
- Interface headers
- Banner areas
- Wide display spaces
- Desktop applications
- Marketing materials

❌ **Don't use for:**
- Small icons
- Avatars
- Compact spaces
- Mobile icons

## Brand Guidelines

**Virgin Media Ireland Colors:**
- **Blue**: #0078D7 (Primary - use for main elements)
- **Yellow**: #FFCC00 (Secondary - use for accents)

**Logo Usage:**
- Use logos on white or light backgrounds for best visibility
- Maintain aspect ratio when resizing - do not distort
- Do not modify colors, rotate, or alter the logo design
- Ensure adequate clear space around logos

**Color Accessibility:**
- Blue (#0078D7) meets WCAG AA standards on white backgrounds
- Ensure sufficient contrast for text and UI elements
